GUIA DE APLICAÇÃO CRUD ANGULAR / SPRING BOOT

Descrição.:

Este projeto demonstra um aplicativo CRUD básico, (Criar, Ler, Atualizar e Excluir). Estas são as funções essenciais de qualquer sistema de armazenamento persistente, permitindo aos usuários gerenciar dados de forma eficaz.

Funcionalidades.:

  • Criar: Adicionar novos registros de dados.
  • Ler: Visualizar ou recuperar registros de dados existentes.
  • Atualizar: Modificar registros de dados existentes.
  • Deletar: Remover registros de dados permanentemente.

Como utilizar.:

Para realizar operações CRUD, siga estas ações básicas:

  1. Criar: Use o formulário para adicionar novas entradas.
  2. Ler: Visualize as entradas exibidas em uma tabela de dados.
  3. Atualizar: Selecione uma entrada e edite seus detalhes.
  4. Deletar: Remova uma entrada permanentemente usando o botão de exclusão.

A aba menu no canto superior esquerdo da bara de navegação oferece o mesmo crud em duas interfaces de dados diferentes, uma utilizando banco de dado relacional, POSTGRESQL, outra utilizando MONGODB.

Tecnologias Utilizadas.:

  1. Angular
  2. Angular Material
  3. Typescript
  4. HTML & CSS

  6. Docker
  7. Postgres
  8. MongoDB
